# 项目简介

“Clubhouse”（俱乐部会馆）的音频聊天应用程序日前火了。这款社交软件的功能相对单一或者说“独特”：一群人线上通话，另一群人听。不过，Clubhouse聊天内容不受限，而且只能受邀才能加入，因此备受职业人士和精英阶层的追捧。有媒体称，它是将现实世界的精英主义搬到了虚拟世界。于是，借此契机，个人打算用flutter开发一款类似聊天软件，欢迎大家关注，一键三连。
